VkGwtdesigner
Welcome to VkGwtDesigner Demo. VkGwtDesigner helps in rapid UI development and can save or load the layout using a load string. Please have a look and leave a comment at Project Site. Please also note that this demo works predictably for firefox 3+ only
Instructions
  1. Right click to start and for options on each widget
  2. Drag the widget to position it on the canvas
  3. To add javascript for event handlers,
    • click on a widget, to popoulate its id in the textarea e.g. (&4)
    • Treat this id as element and invoke javascript functions or attributes e.g. (&4).innerHTML = new Date();
  4. Style toolbar changes context as the widgets are clicked and applies styles to give a WYSIWYG feel.
  5. Events are only recorded in the design mode. To see a working version of your application choose "Quick Preview" from the menu which appears as a result of right click
